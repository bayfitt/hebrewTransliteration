Tips

What’s the best text to use?

Any unicode Hebrew text will work. You can enter the text yourself or copy and paste the text from your favorite online Hebrew Bible.

Some transliterations don’t seem right.

There are a few known issues. Defectively written long vowels are not properly transliterated nor short vowels written using a mater. The qamtas qatan is not always accurate as well.

What should I do if I find something wrong?

Feel free to send me a message through GitHub or Academia.edu.

Can I work on the code myself?

Yes! The code is freely available on GitHub. Hack away!
